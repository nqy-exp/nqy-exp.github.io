**Numbering Convention**: X000-NNNN-LCGZ-00

**Format**: Sample ID-Treatment Mode (Phase 3 only)-Image Structure& mounting medium-Sequence Number

## Sample ID:

- **T**: Test
- **Y**: Sample

## Image Structure & Medium Abbreviations:

- **L**: Longitudinal
- **C**: Cross-sectional
- **G**: Glycerin
- **Z**: Nail base coat
- **N**: Unembedded
